# Corepunk Builder

A fan-made character planner for **Corepunk**. Pick a hero and mastery,
equip weapons, artifacts, chips and runes, choose your talents, and see an
instant stats sheet with a DPS estimate.

- Hero & mastery selection with a kit overview (Q/W/E/R/passive)
- Weapons, artifacts and universal spells, tracking rarity and upgrade state
- Up to 3 talent trees
- Editable base stats and item roll values feed a live stats & DPS panel
- Save builds locally, export/import as JSON, or share a build via link

Static site, no backend -- hosted on GitHub Pages.

Fan-made, not affiliated with or endorsed by the developers of Corepunk.
Values shown are indicative -- synthesis rolls are randomized in-game.
